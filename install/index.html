Before you click install

  1. Use a desktop browser: Chrome, Edge, or Opera. Safari, Firefox, and mobile browsers won't work.
  2. Plug the board into your computer with a USB-C cable that supports data (not power-only).
  3. Click the install button for your board, pick the serial port, then choose Install.

Not sure which version you have? Flashing the wrong one won't brick the board — the e-ink panel just won't draw correctly. Try the other variant if the screen looks wrong after the first boot.
