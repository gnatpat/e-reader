E-ink Reader Installer

Flash the firmware directly from your browser.

Your browser doesn't support Web Serial. Use desktop Chrome, Edge, or Opera. Open this page over HTTPS.

Before you click install

  1. Use a desktop browser: Chrome, Edge, or Opera. Safari, Firefox, and mobile browsers won't work.
  2. Plug the board into your computer with a USB-C cable that supports data (not power-only).
  3. Click Install firmware, pick the serial port, then choose Install.

If the port list is empty, unplug and replug the board, or try a different cable. On Windows you may need to hold the BOOT button while plugging in the first time.
